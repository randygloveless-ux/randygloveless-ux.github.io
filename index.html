Home Assistant

This is a personal home automation application that controls a Nest thermostat via the Google Smart Device Management API. No data is collected, stored, or shared with any third parties. All automation runs locally on a private home network.

Privacy Policy

Privacy Policy

Last updated: June 23, 2026

This application is for personal use only. It uses the Google Smart Device Management (SDM) API to control a Nest thermostat on a private home network.

  • Data collection: No personal data is collected, stored, or transmitted to any third party.
  • Google API usage: OAuth tokens are stored locally on a private server and are used solely to authenticate with the SDM API.
  • Data sharing: No data is shared with anyone.
